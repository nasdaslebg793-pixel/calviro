À propos de Calviro

Un site pour calculer ce dont t'as besoin au quotidien, sans prise de tête.

🎯 Notre mission

Internet regorge de calculateurs en ligne, mais la plupart sont insupportables : 3 popups de pub avant d'arriver au formulaire, demandent ton email avant de te donner le résultat, ou bloquent les fonctionnalités utiles derrière un abonnement. Calviro fait l'inverse : tu arrives, tu calcules, t'as ton résultat. Point.

💡 Notre philosophie

  • Gratuit, vraiment : pas de paywall, pas de "version premium". Les 51 outils sont accessibles sans condition.
  • Aucune inscription : on ne te demande ni ton email, ni ton numéro de téléphone, rien.
  • 100% privé : tous les calculs se font dans ton navigateur. Aucune donnée n'est envoyée sur un serveur.
  • Pas de pub envahissante : aujourd'hui aucune pub. Si on en ajoute un jour ce sera discret et hors du flux de l'outil.
  • Open source : le code est consultable sur GitHub. Tu peux vérifier qu'on ne ment pas sur la privacité.
  • Rapide : aucun chargement lourd, pas de tracker qui ralentit le site. Calviro est conçu pour fonctionner sur n'importe quel appareil, même un vieux téléphone.

📂 Qu'est-ce qu'on trouve sur Calviro ?

Une bibliothèque de calculateurs pour les sujets les plus recherchés du quotidien français :

  • 🏠 Immobilier : simulateur de crédit, capacité d'emprunt, frais de notaire, rentabilité locative, plus-value, DPE, surface habitable, tableau d'amortissement
  • 💰 Finance : impôts, intérêts composés, crédit auto et conso, livret A, TVA
  • 💼 Travail : salaire brut/net, salaire chargé, TJM freelance, chômage, retraite, APL, indemnité de licenciement
  • 🏃 Santé : IMC, calories quotidiennes (TDEE/BMR), eau, allure de course, cycles de sommeil, date d'accouchement et d'ovulation, alcoolémie
  • 📅 Temps : âge précis, jours entre 2 dates, Pomodoro, fuseaux horaires
  • 🛠️ Quotidien : CV, mot de passe, QR code, lien WhatsApp, PDF (fusion, conversion), monnaies, conversion unités, pourcentage, pourboire, quantité peinture
  • 🎓 Études & jeux : Bac, compteur de mots, pseudo, probabilités, calcul mental chronométré, tirage au sort

🔧 Comment c'est construit ?

Calviro est un site 100% statique hébergé chez Vercel. C'est-à-dire qu'il n'y a pas de serveur applicatif, pas de base de données, pas de backend. Tout le code s'exécute dans ton navigateur, en JavaScript pur.

Conséquences :

  • Le site est instantané (aucune attente serveur)
  • Aucune donnée ne quitte ton appareil (sauf si tu utilises explicitement le convertisseur de monnaies qui interroge une API publique de taux de change)
  • Le site peut fonctionner hors-ligne grâce à la PWA
  • Le coût d'hébergement est quasi nul (raison pour laquelle on peut se permettre d'être gratuit)

🛡️ Privacité & sécurité

On prend la privacité au sérieux :

  • Aucun tracker : pas de Google Analytics, pas de Facebook Pixel, pas de Hotjar
  • Aucun cookie tiers : juste les "cookies" techniques (en réalité du localStorage) pour mémoriser ton thème, tes favoris, ton historique — tout reste local
  • HTTPS partout : tes données sont chiffrées en transit
  • Hébergeur fiable : Vercel, l'un des hébergeurs les plus respectueux du standard
  • Code public : si tu doutes, va vérifier sur GitHub. Toute promesse non tenue serait visible.

Pour les détails légaux : politique de confidentialité, mentions légales, CGU.
