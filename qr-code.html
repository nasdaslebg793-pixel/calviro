Qu'est-ce qu'un QR Code ?

Un QR Code (Quick Response Code) est un code-barres 2D qui peut contenir bien plus d'informations qu'un code-barres classique. Inventé en 1994 par l'entreprise japonaise Denso Wave pour l'industrie automobile, il s'est démocratisé partout depuis 2010 avec les smartphones.

L'usage a explosé pendant la pandémie de Covid-19 (menus de restaurant sans contact, pass sanitaire), et il est aujourd'hui devenu un outil banal de la vie quotidienne.

À quoi sert un QR Code ?

  • Site web / URL : la personne scanne et ouvre directement la page
  • Carte de visite : nom, email, téléphone tout en un
  • Wi-Fi public : se connecter sans taper le mot de passe
  • Menus de restaurant : afficher la carte sans support papier
  • Tickets, billets : transport, événements
  • Paiements (PayPal, Lydia, Lyf Pay)
  • Géolocalisation : ouvrir un lieu sur Google Maps
  • SMS pré-rempli ou appel rapide
  • Inscription newsletter
  • Promo / marketing : campagnes avec tracking

Comment fonctionne notre générateur ?

Tape ou colle ton contenu dans le champ, clique sur "Générer". Le QR Code apparaît immédiatement et tu peux le télécharger en PNG. Tout se passe localement dans ton navigateur — ton texte n'est jamais envoyé sur un serveur. Idéal pour des contenus sensibles.

https://calviro.vercel.app

Email pré-rempli

Téléphone

tel:+33612345678

Ouvre l'app téléphone avec le numéro prêt à appeler.

Wi-Fi (très utile)

Format magique : un scan et tes invités se connectent automatiquement à ton Wi-Fi. Très pratique pour cafés, hôtels, soirées chez soi.

Comment scanner un QR Code ?

Aujourd'hui, plus besoin d'application dédiée. Les appareils photo natifs des smartphones modernes (iPhone depuis iOS 11, Android depuis Android 9) détectent automatiquement les QR Codes. Il suffit de :

  1. Ouvrir l'appareil photo
  2. Pointer vers le QR Code
  3. Une notification apparaît avec le contenu
  4. Cliquer dessus pour ouvrir/exécuter

Sécurité : attention aux faux QR Codes

Le "quishing" (QR phishing) est en hausse. Des malfaiteurs collent des faux QR Codes sur des stations de chargement, parkings, affichages publics. Ces codes mènent à des sites frauduleux pour voler tes infos. Précautions :

  • Vérifie l'URL avant de cliquer sur le lien proposé après scan
  • Méfie-toi des QR Codes physiques sur du mobilier public (peuvent être recouverts d'un faux)
  • Ne saisis pas tes infos bancaires sur un site atteint via QR Code suspect
  • Pas de QR Code dans un email non sollicité

Versions et capacité

Les QR Codes ont plusieurs versions selon la quantité d'information à stocker :

VersionCapacité (texte)Usage typique
Version 1~25 caractèresURL courte
Version 10~700 caractèresvCard complète
Version 25~2 500 caractèresLongues données
Version 40 (max)~4 296 caractèresMaximum théorique

Plus tu mets d'infos, plus le QR Code devient dense et difficile à scanner. Pour les URLs, utilise un raccourci (bitly, tinyurl) si l'URL d'origine est très longue.

Niveaux de correction d'erreur

Les QR Codes intègrent un système de correction d'erreur pour rester lisibles même si une partie est abîmée ou tachée :

  • L (Low) : 7 % de récupération
  • M (Medium) : 15 % — notre choix par défaut
  • Q (Quartile) : 25 %
  • H (High) : 30 % — utilisé quand on veut intégrer un logo au centre

Astuces design

  • Contraste élevé : noir sur blanc est optimal. Évite les couleurs trop pâles.
  • Marge blanche autour : indispensable pour que le scanner détecte les bords (au moins 4 modules de blanc)
  • Taille minimum : 2 × 2 cm pour un scan facile à 30 cm, 5 × 5 cm pour un panneau distant
  • Sur fond sombre : tu peux inverser (clair sur foncé) mais certains scanners ont du mal
  • Logo central : possible avec niveau H, mais teste après ajout

QR Code dynamique vs statique

  • QR Code statique (le nôtre) : le contenu est figé dans le code. Si tu changes ton URL, il faut générer un nouveau QR Code.
  • QR Code dynamique (payant, type Bitly, QRCode Monkey) : le QR Code pointe vers une URL intermédiaire qui peut être modifiée. Utile pour le marketing (changer la cible sans réimprimer).

Pour 95 % des usages personnels, le statique suffit largement. Notre outil produit du statique.
